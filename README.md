# Projeto: Sistema de Gerenciamento de Tarefas (To-Do List) ✅

- O projeto consiste no desenvolvimento de uma aplicação fullstack para gerenciamento de tarefas pessoais, permitindo criar, organizar e acompanhar atividades do dia a dia.

## 🎯 Objetivos

1. Criar um sistema que permita o cadastro, consulta, atualização e exclusão de tarefas.
2. Disponibilizar uma API REST desenvolvida com Spring Boot e Java.
3. Construir uma interface web moderna e responsiva com React.
4. Implementar sistema de autenticação de usuários.
5. Organizar tarefas por prioridade e status.

---

## 🛠️ Funcionalidades principais

1. Cadastro e autenticação de usuários.
2. Criação de tarefas com título, descrição e prioridade.
3. Atualização do status das tarefas (pendente, em progresso, concluída).
4. Organização de tarefas por etiquetas (labels).
5. Interface intuitiva para visualização e gerenciamento de tarefas.
6. Validação de dados com feedback visual ao usuário.

---

## 📚 Tecnologias

### Backend

- **Java 17**
- **Spring Boot 3.5.6** (Spring Data JPA, Spring Validation, Spring Web)
- **H2 Database** (banco de dados em memória)
- **Lombok** (redução de código boilerplate)
- **Gradle** (gerenciamento de dependências)

### Frontend

- **React 19**
- **TypeScript**
- **Vite** (build tool)
- **Material-UI (MUI)** (componentes de interface)
- **React Router DOM** (navegação)
- **Axios** (requisições HTTP)
- **TailwindCSS** (estilização)
